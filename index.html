HTML

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
    • To use Git version control so that I can keep track of the styling changes I make to my project in the prework-study-guide GitHub repository, following these steps:
    1. I add, commit,check current branch is up to date, and push up the code in the feature branch.
        -Git commands:
      1. git add -A
      2. git commit -m "added CSS to project"
      3. git pull origin main
      4. git push origin feature/add-css
    2. I open a pull request with my feature branch as the comparison branch and the default branch as the base branch.
    3. I merge the PR so that the main branch holds the updated code for the project.
    4. I update local main branch by navigating to the main branch by entering the following Git command:
      git checkout main
    5. Then pull the remote main branch into the local main branch by entering the following command:
      git pull origin main
